# Orchestration Expert Self-Improvement Triggers

## When to Update Expertise

Update `expertise.yaml` when:

1. **New coordination pattern**
   - Better way to manage agents
   - Add to `patterns`

2. **Error handling improved**
   - Discovered edge case in parallel execution
   - Document recovery pattern

3. **Performance optimization**
   - Faster batch execution
   - Document approach

- confidence >= 0.8
- usage_count >= 10
